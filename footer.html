About

The COVID-19 Real-time Information System for Preparedness and Epidemic Response (CRISPER) has been developed to provide transparent and spatially explicit real-time information for COVID-19 cases and contact tracing locations in Australia. The system comprises four main components:

  • Contact Tracing Map
  • Automatic Alert System
  • Data Visualisation Dashboards
  • Data Engine
This system provides integrated data from multiple sources in one platform which optimises information sharing with public health responders, primary health care practitioners and the general public.
